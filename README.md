# SteamworksTesting
## A testing, debugging, and fun tool for Steam

This tool allows you to give yourself any achievement on any game, it allows to create fake lobbies, find out build info, and so much more! It is intended for debugging, but isn't necessarily not usable for other purposes.

# Install guide:
Install all of the files in the Release, put them in a folder, open SteamworksTest.exe (make sure Steam is open too)
Tada
